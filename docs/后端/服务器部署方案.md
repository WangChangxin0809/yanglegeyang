# 牛马日记 — 服务器部署方案

## 目标服务器

`8.140.194.184`（与金毛大战波斯猫等项目共用）

## 项目占用资源

| 资源 | 位置 / 说明 | 预估占用 |
|------|------------|---------|
| 项目目录 | `/home/work/ai_hustle_diary/server/` | ~50 MB |
| Docker PostgreSQL 容器 | `hustle_diary_pg`（postgres:16-alpine 镜像） | ~250 MB |
| Docker PG 数据卷 | 数据库 `hustle_diary`，2 张表 | <10 MB |
| Redis | 共用已有 Redis（6379），使用 DB 4477 隔离 | 几乎不占 |
| uv + Python 依赖 | `/root/.local/bin/uv` + `.venv/` | ~200 MB |
| systemd 服务 | `/etc/systemd/system/ai_hustle_diary.service` | 1 个文件 |
| **合计新增** | | **~500 MB** |

## 服务器需要修改的地方

### 1. Nginx 配置（已有文件，改一处）

文件：`/etc/nginx/sites-available/game.yunque.co`

```diff
  location /hustle/ {
-     proxy_pass http://127.0.0.1:8066;
+     proxy_pass http://127.0.0.1:8089/;
  }
```

> 多个小游戏共用 `game.yunque.co` 域名，通过路径前缀区分：  
> - `/hustle/*` → 牛马日记（:8089）  
> - 其他游戏保持原有 location 不变

### 2. 新增 Docker 容器

```bash
docker run -d --name hustle_diary_pg --restart=always \
    -e POSTGRES_PASSWORD=<密码> -e POSTGRES_DB=hustle_diary \
    -p 127.0.0.1:5432:5432 postgres:16-alpine
```

### 3. 新增 systemd 服务

```bash
# /etc/systemd/system/ai_hustle_diary.service
ExecStart=uv run uvicorn main:app --host 0.0.0.0 --port 8089
WorkingDirectory=/home/work/ai_hustle_diary/server
```

### 4. 安装 uv（如未安装）

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## 端口分配

| 端口 | 用途 |
|------|------|
| 8089 | 牛马日记后端（FastAPI） |
| 5432 | Docker PostgreSQL（仅 127.0.0.1） |
| 6379 | 共用 Redis，DB 4477 |

## 前端配置

```js
// client/global.js
GameGlobal.serverUrl = 'https://game.yunque.co/hustle'
```

前端请求 `serverUrl + '/api/auth/login'` → Nginx 去掉 `/hustle` 前缀 → 后端收到 `/api/auth/login`，后端代码无需修改。
